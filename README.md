# Gestión de Datos de Países en Python

## Descripción

Este proyecto fue desarrollado como Trabajo Práctico Integrador de la materia Programación 1.

La aplicación permite gestionar información de distintos países utilizando Python, listas, diccionarios, funciones y archivos CSV. El sistema puede realizar búsquedas, filtros, ordenamientos y generar estadísticas a partir de los datos almacenados.

## Funcionalidades

* Agregar países.
* Actualizar población y superficie.
* Buscar países por nombre (coincidencia parcial o exacta).
* Buscar países con o sin acentos.
* Filtrar por continente.
* Filtrar por rango de población.
* Filtrar por rango de superficie.
* Ordenar por nombre.
* Ordenar por población.
* Ordenar por superficie.
* Mostrar estadísticas generales.

## Tecnologías Utilizadas

* Python 3
* CSV
* Listas
* Diccionarios
* Funciones
* Estructuras condicionales y repetitivas
